# Garbage-Classification-using-MobileNetV2
A deep learning project for classifying garbage images into 12 categories using MobileNetV2 and Transfer Learning.
This project implements a garbage image classification 
system using Transfer Learning with MobileNetV2. The model 
is trained on 12,400 images and classifies waste into 12 
categories, including Battery, Biological, Cardboard, Glass, 
Metal, Paper, Plastic, Shoes, and Trash. The dataset is split into 
80% training, 10% validation, and 10% testing 
for model development and evaluation.
